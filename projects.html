soundtrack

CSS animations, Javascript/jQuery music playback

The client for this project was me. The app, soundtrack, is an admittedly sad Spotify-type clone. But that's okay -- I didn't need to create a music player for my music (livin' the Spotify premium life!), I needed an opportunity to grow my development skills and showcase my new knowledge in a handy package.

To view a demo of soundtrack, click here. To check out my code, here's the github repo.

Skills featured in this app:

  • Custom UI design (adapted from a tutorial)
  • Basic page layout in HTMl and CSS
  • CSS/Javascript animations
  • Dynamic content loading with Javascript
  • Refactoring vanilla JS into jQuery (and vice versa)
  • Controlling audio playback with Buzz
